## 1. Node.js ස්ථාපනය කිරීම (Install Node.js)
පහත ලින්ක් එකෙන් Node.js භාගත කර ඉන්ස්ටෝල් කරගන්න.
https://nodejs.org/

## 2. ඇප් එක රන් කිරීම (Running the App)
ඔබ Windows පාවිච්චි කරන්නේ නම්, මෙම ෆෝල්ඩරයේ ඇති `run.bat` ෆයිල් එක මත ඩබල් ක්ලික් කරන්න.
එය ස්වයංක්‍රීයව අවශ්‍ය දේ සකසා ඇප් එක ඕපන් කරනු ඇත.
